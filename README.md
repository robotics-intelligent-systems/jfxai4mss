# Artificial Intelligence for Marine System Simulation

Digital twin of Gunnerus vessel. LOTUSim is a real-time maritime simulation platform for human-vehicle teaming. Wave Energy Converter Simulator. OpenSource Controller (ROSCO) toolbox. Design optimization of floating offshore wind turbine with surrogate models. Mechanical documents of the sailboat. The VesselAI project enables maritime digitization through large-scale analysis, artificial intelligence, and digital twins. The Mobile Ship Bridge is a configurable ship’s bridge system. SIREHNA's Scientific Computing Toolbox. The Marine Systems Simulator (MSS) is a Matlab and Simulink library for marine control systems design. ShipMMG is an unofficial Python package of ship maneuvering simulation. A FreeCAD workbench aiding in naval vessel design. Development of an open-source implementation of the OpenBridge Design System. Modelica library for naval architecture simulations. PathSim is a flexible block-based time-domain system simulation framework. FMI4j is a software package for dealing with Functional Mock-up Units (FMUs). OSP libcosim is a C/C++ library for co-simulation. OSP marine reference models. xdyn is a lightweight ship simulator modelling the dynamic behaviour of a ship at sea. DCPLib is a C++ implementation of the Distributed Co-Simulation Protocol (DCP). Combining Vessel.js with OpenBridge (AHO). Imaging sonar simulation for real-time underwater applications. OceanSim is a GPU-Accelerated Underwater Robot Perception Simulation Framework. A glider flight model for Slocum ocean gliders. An implementation of a Signal K central server for boats. Stonefish is an advanced C++ simulation library designed for marine robotics. HoloOcean enables easy simulation of marine robotics and autonomy with a wide variety of sensors, agents, and features. OpenSSN is a submarine simulation. Real-time 3D submarine simulation. Aquabot is a self-Improving Autonomous Underwater Manipulation. OpenROV is a DIY telerobotics community centered around underwater exploration & adventure

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
